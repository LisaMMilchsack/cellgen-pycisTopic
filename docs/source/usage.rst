Installation
------------

To install pycistarget

.. code-block:: console

   (.venv) $ git git clone https://github.com/aertslab/pycisTopic.git
   (.venv) $ cd cd pycisTopic
   (.venv) $ pip install -e .

Check your pycistarget version by running:

>>> import pycisTopic
>>> pycisTopic.__version__
